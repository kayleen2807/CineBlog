# CineBlog

Repositorio para trabajar con  el proyecto (en comienzo)
## ¿Qué CineBlog?
Este proyecto consiste en el desarrollo de un sitio web de reseñas de películas y series, diseñado para que los usuarios puedan compartir opiniones y conocer las recomendaciones de otras personas sobre distintos contenidos audiovisuales.
La plataforma permite que los usuarios se registren y creen una cuenta para poder publicar comentarios, escribir reseñas y participar activamente dentro del sitio. De esta forma, se fomenta la interacción entre los usuarios y el intercambio de opiniones sobre películas y series.
Además, el sistema también ofrece la posibilidad de navegar como visitante, lo que permite a cualquier persona explorar el contenido del sitio, leer reseñas y consultar opiniones sin necesidad de registrarse.
