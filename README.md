# 🍿 Cineblog

Sitio web de reseñas de películas y series, diseñado para que los usuarios puedan compartir opiniones y conocer las recomendaciones de otras personas sobre distintos contenidos audiovisuales.


## 🧠 Descripción

La plataforma permite que los usuarios se registren y creen una cuenta para poder publicar comentarios, escribir reseñas y participar activamente dentro del sitio. De esta forma, se fomenta la interacción entre los usuarios y el intercambio de opiniones sobre películas y series. Además, el sistema también ofrece la posibilidad de navegar como visitante, lo que permite a cualquier persona explorar el contenido del sitio, leer reseñas y consultar opiniones sin necesidad de registrarse.

El sistema permitirá a los usuarios:

• Registrarse y crear una cuenta

• Publicar reseñas de películas o series

• Comentar y compartir opiniones

• Explorar recomendaciones de otros usuarios

También existirá la opción de navegar como visitante, lo que permitirá leer reseñas y consultar opiniones sin necesidad de registrarse.

⚠️ Nota: Este proyecto actualmente se encuentra en desarrollo, por lo que algunas funcionalidades aún no están implementadas.
